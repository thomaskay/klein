h1. Klein

h2. Description

Klein is a lightweight Rails I18n database backend.

h2. Setup

h3. Installation

<pre><code>
script/plugin install git@github.com:thomaskay/klein.git
</code></pre>

h3. Setup migrations

Create and run migrations with:
<pre><code>
rake klein:db:create
rake db:migrate
</code></pre>

h3. Define backend

And add the following to your ApplicationController:
<pre><code>
I18n.backend = I18n::Backend::KleinBackend.new  
I18n.default_locale = :en
</code></pre>
